# OpenRelief Synthetic Data License

## Scope

This file covers synthetic letters, survivor profiles, eval fixtures, screenshots, and demo case data committed to OpenRelief.

## License Grant

Synthetic data in this repository is dedicated to the public domain under CC0 1.0 Universal where legally possible. Where public domain dedication is not recognized, contributors grant permission to use, copy, modify, publish, distribute, and test the synthetic data for any purpose without payment.

## No Real Survivor Data

No real survivor data may be committed. Do not include real names, addresses, phone numbers, email addresses, agency account IDs, Social Security numbers, immigration details, medical records, insurance claim numbers, photos, letters, screenshots, or copied case notes.

## Contributor Rules

- Use fictional people, fictional addresses, and synthetic agency text.
- Keep policy claims source-backed by public official sources.
- Mark OCR noise, adversarial text, and high-risk scenarios as synthetic.
- Avoid content that could identify a real person, household, case worker, or agency account.
- Remove any accidental real PII before opening a pull request.

## Warranty

Synthetic data is provided as-is for testing and documentation. It is not legal advice, government guidance, eligibility review, or disaster assistance instruction.
